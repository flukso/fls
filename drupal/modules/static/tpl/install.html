Step 1: Configuring wifi & sensors

Power up the FLM. Connect your computer to the FLM's ethernet port. Surf to http://192.168.255.1. Configure the wireless interface with the proper name and security key. The globe LED should light up. Proceed to the sensor page and configure each of the ports you wish to use on the FLM.

Step 2: Securing the fluksometer

Disconnect all cables from the FLM. Find a suitable location near the fuse box to install the FLM. Secure the FLM with a plastic cable tie or by using the rear mounting holes.

Step 3: Attaching the current clamp

For safety reasons, switch off the main electricity supply. Within the fuse box, attach a current clamp to each of the live wires running from the electricity meter. Close the clamp firmly. You should hear a double click. The lip should lie flush with the clamp's body.

For safety reasons, the live wire should be fully insulated. Never attach the clamp to a non-insulated live wire!

Step 4: Connecting the clamp's snsor cable

Connect the cable from the current clamp to one of the FLM's analog input ports. Take care of connecting the black and red cable as shown in the drawing on the left.

Step 5: Powering up

Switch the main electricity supply back on. Activate your FLM by connecting the power plug. Visit https://www.flukso.net/user/register to create an account. Then click the My account -> devices tab to assign the FLM's serial to your account. A first reading will be visible on the hour chart within five minutes from powering up.
