STEP 1: CONFIGURING WIFI & SENSORS

 

Power up the Fluksometer. Connect your computer to the Fluksometer's ethernet port. Surf to http://192.168.255.1. Configure the wireless interface with the proper name and security key. The globe LED should light up. Proceed to the sensor page and configure each of the ports you wish to use on the Fluksometer.

STEP 2: SECURING THE FLUKSOMETER

 

Disconnect all cables from the Fluksometer. Find a suitable location near the fuse box to install the Fluksometer. Secure the Fluksometer with a plastic cable tie or by using the rear mounting holes.

STEP 3: ATTACHING THE CURRENT CLAMP

 

For safety reasons, switch off the main electricity supply. Attach a current clamp to each of the live wires running from the electricity meter to the fuse box. Close the clamp firmly. You should hear a double click. The lip should lie flush with the clamp's body.

STEP 4: CONNECTING THE CLAMP'S SENSOR CABLE

 

Connect the cable from the current clamp to one of the Fluksometer's analog input ports. Take care of connecting the black and red cable as shown in the drawing on the left.

STEP 5: POWERING UP

 

Switch the main electricity supply back on. Activate your Fluksometer by connecting the power plug. Visit https://www.flukso.net/user/register to create an account. Then click the My account -> devices tab to assign the Fluksometer's serial to your account. A first reading will be visible on the hour chart within five minutes from powering up.
